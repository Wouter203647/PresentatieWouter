# Ontwikkelreis Wouter Malgo

Deze map bevat een statische website die je kunt publiceren via GitHub Pages.

## Bestanden

- `index.html`: de inhoud van de webpresentatie
- `style.css`: de vormgeving en kleurcodes
- `script.js`: navigatie, toetsenbordbediening en leesmodus

## Uploaden naar GitHub Pages

1. Maak een nieuwe repository aan op GitHub.
2. Upload deze drie bestanden naar de root van de repository.
3. Ga naar Settings.
4. Ga naar Pages.
5. Kies Deploy from a branch.
6. Kies branch `main` en folder `/root`.
7. Sla op.

Na publicatie krijg je van GitHub een URL naar je website.

## Bediening

- Pijltje rechts of spatie: naar het volgende onderdeel
- Pijltje links: terug
- Knoppen onderin: verhalend navigeren
- Knop rechtsboven: wisselen tussen presentatie en leesmodus


## Aanpassing

Deze versie bevat de aangepaste slottekst en compactere responsive styling voor presentatieschermen zoals 1920 x 1080.
